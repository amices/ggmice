# Package index

### Core function

- [`ggmice()`](http://amices.org/ggmice/reference/ggmice.md) : Plot
  incomplete or imputed data

### Other plotting functions

- [`plot_corr()`](http://amices.org/ggmice/reference/plot_corr.md) :
  Plot correlations between (incomplete) variables
- [`plot_flux()`](http://amices.org/ggmice/reference/plot_flux.md) :
  Plot the influx and outflux of a multivariate missing data pattern
- [`plot_miss()`](http://amices.org/ggmice/reference/plot_miss.md)
  **\[experimental\]** : Plot missingness in a dataset
- [`plot_pattern()`](http://amices.org/ggmice/reference/plot_pattern.md)
  : Plot the missing data pattern of an incomplete dataset
- [`plot_pred()`](http://amices.org/ggmice/reference/plot_pred.md) :
  Plot the predictor matrix of an imputation model
- [`plot_trace()`](http://amices.org/ggmice/reference/plot_trace.md) :
  Plot the trace lines of the imputation algorithm

### Old friends

- [`bwplot()`](http://amices.org/ggmice/reference/bwplot.md) :
  Box-and-whisker plot of observed and imputed data
- [`densityplot()`](http://amices.org/ggmice/reference/densityplot.md) :
  Densityplot of observed and imputed data
- [`stripplot()`](http://amices.org/ggmice/reference/stripplot.md) :
  Stripplot of observed and imputed data
- [`xyplot()`](http://amices.org/ggmice/reference/xyplot.md) :
  Scatterplot of observed and imputed data

### Internal functions

- [`theme_mice()`](http://amices.org/ggmice/reference/theme_mice.md) :

  Theme for [mice::mice](https://amices.org/mice/reference/mice.html)
  style
  [ggplot2::ggplot](https://ggplot2.tidyverse.org/reference/ggplot.html)
  objects

- [`theme_minimice()`](http://amices.org/ggmice/reference/theme_minimice.md)
  :

  Minimal theme for
  [mice::mice](https://amices.org/mice/reference/mice.html) style
  [ggplot2::ggplot](https://ggplot2.tidyverse.org/reference/ggplot.html)
  objects
